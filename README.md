# Society of Indian Grandparents (SIG)

Welcome to the Society of Indian Grandparents - a community initiative connecting Indian elderly people in Toronto.

## About

SIG provides resources, events, and support to help Indian grandparents navigate life in a new country and build meaningful connections with others in their community.
